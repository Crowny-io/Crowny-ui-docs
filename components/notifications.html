Markup #

Type classDescriptionUsage
sectionalA sectional notification is a notification that is only located within the content area of a page, these notifications are to inform the user in the section that they are interacting with. E.g. if a user is changing properties of a loyalty program and some changes cannot be changed due to their subscription package.class="sectional"
toastA toast notification is a notification that is used as feedback on interactions by the user. These notifications are only visible for a short amount of time to inform that an action was done successfully. E.g. A user has made changes to their loyalty program settings in the Portalclass="toast"
globalA global notification is a notification that is used to inform the user with some sense of urgency. These notifications are always on top of the content area of a page and should try to attract the attention of the user. E.g. Crowny's ToS have changed.class="global"
Color classDescriptionUsage
warningWarning is used in order to display a sense of urgency. It is not critical, but is important to read.class="warning"
errorError is used when something is urgent. This means that that information that is displayed is critical and should be read.class="error"
infoInfo is used to inform the user, this is a good to know type of information and there is no sense of urgency in the information that is displayed.class="info"
info-reversedInfo-reversed is used for darkmode settingsclass="info-reversed"
Icon classDescriptionUsage
with__iconThe icon is to emphasise the type of information you want to supply with the notification, aligns left by default. E.g. have an icon with a cross when you get an error for exampleclass="with__icon"
with__icon__rightThe icon here is used for a replacement of the text CTA. E.g. whenever a user is allowed to close the notification, aligns on the right.class="with__icon__right"
