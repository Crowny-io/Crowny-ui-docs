Spacers #

Spacers #

Spacers are pre-defined from within the design system, Crowny-UI gives you an array of spacers to use with paddings or margins.

Important

Currently there's no spacer classes available vars. namespacing is required.

Options #

OptionPX Amount
spacer-22
spacer-44
spacer-88
spacer-1212
spacer-1616
spacer-2424
spacer-3232
spacer-4848
spacer-6464
spacer-7272

Usage #

.spacer__example {
    padding: vars.$spacer-4;
    margin: vars.$spacer-16;
}
