# Tabs

# Default Tabs

The tab button is used wherever the user needs to choose between different content types or make a visual selecten between a different radio values.

# Preview

Important

The crowny__toggles__tabs can support up to a maximum of 6 tab items!

# Markup

<div class="crowny__toggles__tabs">
    <input id="radio-1" type="radio" value="select an op" name="radio-group" checked>
    <label class="tab" for="radio-1"><i class="icon-QR"></i> Black</label>

    <input id="radio-2" type="radio" value="between these" name="radio-group">
    <label class="tab" for="radio-2"><i class="icon-QR"></i> White</label> 

    <input id="radio-3" type="radio" value="between these" name="radio-group">
    <label class="tab" for="radio-3"><i class="icon-QR"></i> Color</label>      

    <div class="crowny__glider"></div>
</div>
