Radios #

Default radios #

The radio button is used wherever the user needs to choose between a group of values.

Preview #

Important

The crowny__toggles comes default with the Crowny-UI library and is needed for the custom checkbox positioning.

Markup #

<div class="crowny__toggles">
    <input id="radio-1" type="radio" value="1" name="radio-group" checked>
    <label for="radio-1">Pick me!</label>
    <input id="radio-2" type="radio" value="1" name="radio-group">
    <label for="radio-2">NO pick me!</label>    
</div>
