# Getting started

# 1. Install Crowny-UI

Installing the Crowny-UI library will grant you all of the styling needed for Crowny related projects.

Login required!

Because Crowny-UI is a private package, you need to login via npm in order to install this package. Run npm login and login to your Crowny Npm account.

npm i @crowny/crowny-ui

# 1.1 Using Crowny-UI

Crowny-UI is a ready to use style library, the package comes with pre-styled:

  • Inputs
  • Buttons
  • Toggles
  • Textual
  • Cards
  • And more..

Start using Crowny-UI by adding the global stylesheet (style.scss) to Vue's loaderOptions in vue.config.js located in your Vue project root folder.

TIP

Additionally we provide a reset.scss that you can use to reset all of the default element styling. Make sure the reset.scss is loaded before anything else, see the code preview below.







Â 







module.exports = {    
    css: {
        sourceMap: true,
        loaderOptions: {
            sass: {
                prependData: `
                    @import "@crowny/crowny-ui/scss/reset.scss";
                    @import "@crowny/crowny-ui/scss/style.scss";
                `,
            },
        },
    },
}

# 2. Install Crowny-Icons

Installing the Crowny Icon pack will grant you all of the icons needed for Crowny related projects.

Login required!

Because Crowny-UI is a private package, you need to login via npm in order to install this package. Run npm login and login to your Crowny Npm account.

npm i @crowny/crowny-icons

# 2.1 Using Crowny-Icons

Crowny-Icons is a ready to use icon-font, the package comes with

  • variables.scss
  • style.scss
  • /fonts
  • style.css

For compatibility reasons we suggest you use the style.css in your projects.

In order to get the Crowny-Icons in your Vue project, you need to include the style.css globally. Open up Vue's main.js located in the root folder and require the style.css.

require('@crowny/crowny-icons/iconpack/style.css')

TIP

Adding @ makes sure Webpack resolves your package from node_modules/{your__package__name}
