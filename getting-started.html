# Getting started

# 1. Install

Install the projects dependencies and node modules via npm.

    npm i

# 2. Run the project [Crowny UI]

Open the projects root folder and run the following command to get the project up and running for development.

    npm run serve

This will open up http://localhost:8080/ by default and should watch/compile any changes you make in the Vue components located in ~/src/components or any of the the SCSS stylesheets located in ~/src/scss/.

# 3. Run Vitepress project

Open the projects root folder and run the following command to get the project up and running for development.

    npm run docs:dev
